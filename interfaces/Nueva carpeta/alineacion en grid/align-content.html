Ejemplo de CSS Grid: align-content

align-content: start

Alinea los grid tracks al inicio del eje vertical.

3
4

align-content: center

Centra los grid tracks verticalmente en el contenedor.

A
B
C

align-content: end

Alinea los grid tracks al final del eje vertical.

I
II
III
IV

align-content: stretch

Estira los grid tracks para llenar el contenedor verticalmente.

Alpha
Beta
Gamma
Delta

align-content: space-between

Distribuye los grid tracks con espacios iguales entre ellos.

X
Y
Z
W

align-content: space-around

Distribuye los grid tracks con espacios iguales alrededor de ellos.

Uno
Dos
Tres
Cuatro

align-content: space-evenly

Distribuye los grid tracks con espacios iguales entre ellos y los bordes del contenedor.

B
C
D
