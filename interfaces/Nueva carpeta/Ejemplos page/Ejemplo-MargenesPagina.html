Ejemplo 2: Configuración de Márgenes de Página

En este ejemplo, utilizamos la regla @page en CSS para establecer márgenes personalizados en todas las direcciones (superior, inferior, izquierda y derecha) de la página al imprimir. Hemos configurado márgenes iguales de 3 centímetros en cada lado.

Para ver los efectos de esta configuración, sigue las instrucciones a continuación para acceder a la Vista Previa de Impresión en tu navegador.
