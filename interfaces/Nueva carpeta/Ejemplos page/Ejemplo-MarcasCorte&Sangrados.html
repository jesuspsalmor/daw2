Ejemplo 5: Agregar Marcas de Corte y Sangrado

En este ejemplo, utilizamos la regla @page en CSS para agregar marcas de corte (crop) y registro (cross), así como para definir un área de sangrado (bleed) en la página impresa. Estas características son esenciales para impresiones profesionales, ya que facilitan el proceso de recorte y aseguramiento de que el contenido se extienda más allá de los bordes finales del documento.

Para ver los efectos de esta configuración, sigue las instrucciones a continuación para acceder a la Vista Previa de Impresión en tu navegador.
