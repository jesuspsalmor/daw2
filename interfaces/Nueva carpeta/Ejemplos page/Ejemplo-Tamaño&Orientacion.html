Documento de Ejemplo: Tamaño y Orientación de la Página

Introducción

Este documento sirve como un ejemplo para demostrar cómo utilizar la regla @page en CSS para establecer el tamaño y la orientación de una página al imprimir. En este caso, configuraremos el tamaño de la página a A4 con una orientación horizontal (landscape).

Para ver los efectos de esta configuración, sigue las instrucciones a continuación para acceder a la Vista Previa de Impresión en tu navegador.
